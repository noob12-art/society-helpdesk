# Society Helpdesk Starter

This is a simple Bootstrap-based complaint management website created from your PDF idea.

## Included Pages

- `index.html`
- `login.html`
- `sign-up.html`
- `sign up.html` redirect helper
- `raise.html`
- `dashboard.html`
- `admin.html`

## Included Folders

- `css/style.css`
- `firebase/firebase-config.js`
- `js/auth.js`
- `js/complaint.js`
- `js/admin.js`

## What Works Now

- Bootstrap-based responsive UI
- resident sign up
- resident login
- admin login
- complaint submission
- complaint tracking dashboard
- admin complaint status update
- shared connection between all files

## Firebase Setup

Open `firebase/firebase-config.js` and replace these placeholder values:

- `YOUR_API_KEY`
- `YOUR_AUTH_DOMAIN`
- `YOUR_PROJECT_ID`
- `YOUR_STORAGE_BUCKET`
- `YOUR_MESSAGING_SENDER_ID`
- `YOUR_APP_ID`

The project is built with a local demo fallback using `localStorage`, so it can still work before Firebase is fully connected.
